# cow4-ai
Artificial Intelligence for Code Of War 2015

https://github.com/damoebius/Cow4

You need NodeJS

### Installation

```sh
$ npm install
```

### Running server

```sh
$  npm run serve
```

### Running client

```sh
$ node client/client.js
```

You need 2 clients for playing game

### Tests
   
```sh
$ mocha tests
```

### Map visualisation
   
```sh
$ node vizualiazer/iadebug.js
$ chrome http://localhost:4000
```
